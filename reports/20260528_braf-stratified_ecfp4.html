5. How to read this report

  1. Performance section: only compare across runs that share the same OOF protocol. Within-fold AUC is not the same number.
  2. SHAP features: these are statistical attributions. A high mean-|SHAP| means "the model relies on this feature", not "this feature is causal".
  3. SAE diagnostics: dead_ratio is the canary. If it climbs above 0.2, the SAE is broken and any feature-level interpretation built on top of it is suspect.
  4. MMP rules: read the CI before the mean. A wide CI (e.g. [-0.05, +0.30]) is not a rule, it is a coincidence with a single exemplar.
